HTML

  • HTML stands for Hypertext Markup Language
  • HTML is used for framing the structure of a webpage and is made up of elements.
  • The head element contains information about the webpage. Ex. metadata
  • The body element represents the visible content shown to the user.
  • Elements are a type of content on a webpage.
  • Examples of elements are text, image, header and list elements
  • Elements can also be used to divide the content on the page and to describe the content to browseres and other developers

CSS

  • CSS stands for Cascading Style Sheets
  • CSS adds style to our webpages
  • CSS defines colors, aesthetics, visual layout and organization
  • There are 3 ways to style a webpage using CSS: Inline, Internal and External
  • Inline- CSS styling an element directly in HTML
  • Internal- CSS style rules for the page in header section that applie to all tagged elements
  • External - Keeps all CSS rules in seperate file and is linked in header (Most common)
  • A margin indicates how much space we want around the outside of an element
  • A padding indivates how much space we want around the content inside an element
